get_threads(pid)

source code  
Obtiene los PIDs de los threads que posee el proceso identificado con
el PID pid. Esta informacion se obtiene de /proc/<pid>/task/
    @pid: Pid del proceso
Devuelve una lista con los PIDs de sus hebras, incluido el suyo.
