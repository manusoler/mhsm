num_cpus()

source code 

Obtiene el numero de CPU's que existen en el sistema mirando en el archivo /proc/cpuinfo. Similiar a lo que obtendriamos de ejecutar el comando 'grep processor /proc/cpuinfo | wc -l'.

Returns: int
Numero de cpus que tiene la maquina

cpuinfo(num_cpu=0)

source code 

Obtiene informacion de la cpu.

Parameters:
  • num_cpu (int) - Numero de la CPU
Returns: list
None o [num_cpu, fabricante, familia, modelo, stepping, frecuencia, cache] donde:
  • num_cpu: Numero de procesador
  • fabricante: Fabricante
  • familia: Familia de la CPU
  • modelo: Modelo
  • stepping: Stepping
  • frecuencia: Frecuenca Mhz
  • cache: Cache
