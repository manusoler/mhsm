leer_info(archivo, num_reg)

source code 

Obtiene la toda la informacion de la BD sobre un registro.

Parameters:
  • archivo (string) - Archivo en el que se encuentra la BD
  • num_reg (int) - Numero de registro que se quiere leer
Returns: tuple
Tupla de (cpus, proc, hebras, mensaje) donde
  • cpus: Es una lista de objetos cpuinfo.CPU
  • procesos: Es una lista de objetos procinfo.Proceso
  • hebras: Una lista con los objetos procinfo.Hebra del proceso proc.
  • mensaje: Es una cadena de texto con informacion acerca del registro, puede estar vacia

get_num_threads_by_state(archivo, state, cpu=-1, frame=-1)

source code 

Obtiene el numero de threads en el estado especificado en la cpu en el frame especificado.

Parameters:
  • archivo (string) - Archivo en el que se encuentra la BD
  • state (char) - Indica el estado de los threads. Puede tomar uno de los siguientes valores:
    • 'R': Running
    • 'S': Sleeping
    • 'D': Waiting
    • 'Z': Zombie
    • 'T': Stopped
    • 'W': Paging]
  • cpu (int) - Indica la cpu de la que se quieren obtener el numero de threads Si cpu = -1 se obtienen los threads de todas las cpus
  • frame (int) - Indica el frame del cual se quieren obtener los thread. Si frame = -1 se devuelven todos los threads de la monitorizacion para la cpu especificada
Returns: int
Numero de threads en el estado 'state' que han estado en la cpu 'cpu' en el frame 'frame'
