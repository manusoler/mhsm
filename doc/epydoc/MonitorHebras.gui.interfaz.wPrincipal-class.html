actualizar_configuracion(self, procesos, colores, log_path, t_refresco, lang, tmp_rep, comenz_rep, sort_by)

source code 

Actualiza los atributos de la configuracion

Parameters:
  • procesos (list) - Una lista con los nombres de los procesos que se quieren monitorizar.
  • colores (list) - Una lista con los colores para cada uno de los estados de un proceso.
  • log_path (string) - Ruta del archivo de log
  • t_refresco (float) - Tiempo en el que se van monitorizando los procesos
  • lang (string) - Lenguaje del programa. Puede ser
    • "Español"
    • "English"
  • tmp_rep (float) - Tiempo de visualizacion entre registros en la pantalla de reproduccion
  • comenz_rep (int) - Modo en el que se comienza la reproduccion:
    • siempre por el primer registro (PRIMERO - 0)
    • siempre por el actual (ACTUAL - 1)
    • se pregunta al reproducir (PREGUNTAR - 2)
  • sort (int) - El orden que llevaran las hebras al mostrarse en la lista:
    • SORT['PID']
    • SORT['NOMBRE']
    • SORT['ESTADO']
Returns: void
