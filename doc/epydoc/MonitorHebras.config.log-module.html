escribir_log(info_cpus, procesos, hebras, archivo, n_log=50)

source code 

Guarda la informacion recogida en un archivo para poder mostrarla mas tarde poco a poco.

Parameters:
  • info_cpus (list) - Lista con objetos CPU, cada uno de ellos con la información de una CPU.
  • procesos (list) - Lista de objetos procinfo.Proceso con la informacion de los procesos principal.
  • hebras (list) - Lista de objetos procinfo.Hebra con la información de cada hebra del proceso.
  • archivo (string) - Archivo en el cual guardar la informacion.
  • n_log (int) - Numero de registros guardados.
Returns: void

leer_log(archivo, num_reg)

source code 

Lee del archivo de log el registro num_reg.

Parameters:
  • archivo (int) - Archivo en el cual guardar la informacion.
  • num_reg (int) - Numero de registro a leer.
Returns: tuple
Una tupla (cpus, proc, hebras) donde:
  • cpus: Es una lista de objetos cpuinfo.CPU
  • procesos: Es una lista de objetos procinfo.Proceso
  • hebras: Una lista con los objetos procinfo.Hebra del proceso proc.
